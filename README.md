# H Themes — WP Theme Store (Next.js + Tailwind)

Project sẵn sàng chạy: demo lưu kho theme WordPress, multi-demo per theme, global search, export/import, fullscreen preview, admin-mini.

## Chạy
1. Cài dependencies:
```bash
npm install
```
2. Chạy dev:
```bash
npm run dev
```
Mở http://localhost:3000

## Deploy
Push lên GitHub và import trên Vercel — Next.js sẽ tự build.
